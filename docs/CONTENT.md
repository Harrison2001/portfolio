# Portfolio Content Guide

This guide explains how to update the content displayed on the portfolio site. All editable text and structured data lives in the `data/` directory.

## Featured Projects

**File:** `data/projects.ts`

Each project is an object with these fields:

| Field | Type | Description |
|-------|------|-------------|
| `title` | string | Project name |
| `type` | string | Category label (e.g. "Full-Stack Product") |
| `description` | string | Short summary for the card |
| `stack` | string[] | Technology tags |

**Current projects:**

1. **Line Dancing Social App** — Full-stack social platform for line dancers
2. **Streaming Analytics Platform** — MovieLens / TMDb analytics with ML and Power BI
3. **Game Pricing Intelligence API** — FastAPI Steam pricing analysis

**To add a project:**

```typescript
{
  title: "My New Project",
  type: "Backend / API",
  description: "What the project does and why it matters.",
  stack: ["Python", "FastAPI", "PostgreSQL"],
},
```

## Experience Timeline

**File:** `data/experience.ts`

Each entry has:

| Field | Type | Description |
|-------|------|-------------|
| `year` | string | Year or range (e.g. `"2026"`) |
| `title` | string | Role or milestone |
| `company` | string | Organization name |
| `description` | string | Brief summary |

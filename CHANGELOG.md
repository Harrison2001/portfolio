# Changelog

All notable changes to this portfolio project are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Documentation

- Added comprehensive README replacing `create-next-app` boilerplate
- Added `docs/` directory with setup, structure, content, deployment, and known issues guides
- Added this changelog

### Known (not yet fixed)

- Page metadata uses placeholder values
- Navbar anchor links for Experience and Skills sections are broken
- Project cards lack demo and repository links
- No mobile navigation menu

## [0.1.0] - 2026-06

### Added

- Initial portfolio site built with Next.js 16, React 19, TypeScript, and Tailwind CSS 4
- Hero section with Framer Motion animations (grid background, aurora glow, particles)
- Fixed navbar with scroll-aware background blur
- Technical Universe skills section (`SkillOrbit`)
- Featured projects grid with three projects (Line Dancing App, Streaming Analytics, Game Pricing API)
- Experience timeline with FGCU research, personal projects, and education
- Engineering focus cards (Data Engineering, Full Stack, Cloud & Infrastructure)
- System design / architecture layers section
- Contact footer with email, GitHub, and LinkedIn links
- Content data modules in `data/` for projects, skills, experience, and domains
- Geist Sans and Geist Mono fonts via `next/font`
